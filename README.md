## 🚀 About the Project

**Digital Tools** is a visually stunning and fully responsive digital marketplace interface built using React. It combines elegant design with practical functionality, offering users a smooth and engaging browsing experience.

From exploring high-quality digital products to managing purchases effortlessly, every interaction is designed with usability and performance in mind. The UI is inspired by professional Figma designs and implemented with pixel-perfect precision to deliver a premium feel.

---

## 🛠️ Technologies Used

- **Framework:** React.js
- **Styling:** Tailwind CSS & DaisyUI
- **Icons:** Lucide React
- **Notifications:** React-Toastify
- **Data Handling:** JSON-based product system
- **Language:** JavaScript (ES6+)

---

## 🌟 Key Features

### 🧩 Dynamic Product Catalog

- Showcases a curated collection of digital products in a clean, responsive 3-column grid layout.
- Each product card dynamically displays tags like **“Popular”** or **“Best Seller”**, along with features and pricing details.
- Powered by a flexible JSON data structure for easy scalability.

---

### 🛒 Real-Time Cart Experience

- Fully functional cart system with live updates.
- Instantly reflects item count in the Navbar.
- Users can add/remove products and view a real-time total price calculation.
- Smooth and intuitive checkout interaction.

---

### 🎨 Interactive UI/UX

- Seamless toggle between **Product View** and **Cart View**.
- Fully responsive design optimized for all devices.
- Instant feedback with toast notifications for actions like **Add**, **Remove**, and **Checkout**.
- Clean, modern interface focused on user experience.
